Minifier + Refactor + Minifier (08th Aug)


    Columns
    Prep:  The name of the minifier/pre-processor
    Mnfy:  The size removed by the minifier
    Trm:   The size removed after the 1st iteration
    Itr:   The number of iterations run within the timeout
    Rfrt:  The size removed by the tool
    Diff1: The size removed by the minifier and the tool (= Mnfy + Rfrt)
    Diff2: The size removed by applying the minifier (if any) again after refactoring
    Remarks
    1. Percentages of sizes are computed w.r.t. the original file sizes.
    2. Sizes here are physical sizes, not those listed in timings.txt.
